# GoTrim

GoTrim is a simple tool to shorten URLs.

### Prerequisites

- Go 1.20 or later

### Installation
1. Clone the repository
    ```sh
    git clone https://github.com/NickNaskida/GoTrim.git
    ```
2. Build the project
    ```sh
    go build
    ```
3. Run the project
    ```sh
    ./GoTrim
    ```

## Development
### Running the tests
```sh
go test -v ./tests
```

### Building the project
```sh
go build
```
